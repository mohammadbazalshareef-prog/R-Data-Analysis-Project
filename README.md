# Student Performance Data Analysis (R)

This project explores student performance data using R, focusing on correlation analysis and data visualization. Developed as part of the **Tuwaiq Academy** R programming course.

# R-Data-Analysis-Project
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1gkMuvzmwV2KD01xIMh67EllNIFGysmE9)

## Features
- **Data Cleaning:** Preprocessing student metrics.
- **Correlation Matrix:** Identifying key factors affecting grades (G1, G2, G3).
- **Visualization:** Professional plots using `ggplot2` and `corrplot`.
